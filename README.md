C++ Mini Projects Collection

A collection of beginner-friendly C++ projects created to strengthen programming fundamentals and problem-solving skills.

## 📌 Projects Included

### 1. 🧮 Scientific Calculator
A menu-driven calculator that performs basic and scientific operations.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- Power
- Square Root
- Factorial

**Concepts Used**
- Switch Case
- Loops
- Functions
- `<cmath>` Library
- User Input/Output
